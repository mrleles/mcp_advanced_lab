# Advanced MCP Applications with Streamable HTTP, Roots, and Sampling

 In this lab, you'll build an advanced MCP application using HTTP transport for remote connectivity, filesystem roots for security boundaries, and sampling for server-initiated LLM requests. You'll create a base HTTP client with HTTP protocol support, then build two applications on top of it: a GUI client app for manual interaction and an AI-powered host app with full remote MCP capabilities.
